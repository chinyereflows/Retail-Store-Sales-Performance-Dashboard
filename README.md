# Retail-Store-Sales-Performance-Dashboard
Transforming raw transactional data into actionable insights for retail optimization.
### Background
This project was developed to gain insights into retail store performance, including year-over-year growth, total revenue, and overall sales trends. The dashboard is designed to help business owners and stakeholders make informed, data-driven decisions based on clear and interactive sales insights.
### Dataset
A retail store sales dataset containing 12,576 rows of transactional data covering the period 2022–2025.
### Tools Used
- Microsoft Excel
### Project Objectives
The objectives of this project are to:
- Analyze sales performance and revenue trends across multiple years
- Identify top-selling products and categories
- Track the number of orders generated
- Evaluate year-over-year revenue growth
- Support strategic business decision-making through data insights
### Dataset Description
The dataset includes the following key fields:
- Category
- Item
- Price per Unit
- Quantity
- Total Spent
- Payment Method
- Location
- Transaction Date
### Tools & Skills Applied
- Microsoft Excel
-	Data Cleaning and Formatting
-	Calculated Fields
-	Pivot Tables and Pivot Charts
-	Dashboard Design and Layout
### Data Cleaning & Preparation
The following data preparation steps were carried out:
-	Removed duplicate records
-	Filled blank cells where values could be logically derived
-	Removed blank rows that could affect analysis accuracy
-	Standardized date, number, and currency formats across the dataset
### Key Metrics Tracked
-	Total Revenue
-	Revenue by Category
-	Year-over-Year Growth
-	Average Order Value (AOV)
-	Number of Items Sold
-	Revenue by Location
-	Number of Orders
-	Preferred Payment Method
-	Monthly Revenue Generated
### Dashboard Features
-	Interactive filters by month, year, and category
-	KPI cards for quick performance summaries
-	Visual charts for trend analysis and comparisons
-	Clean, intuitive, and user-friendly layout
### Insights & Use Cases
-	Identify top-performing product categories
-	Track revenue generation across sales locations
-	Analyze monthly trends to assess and predict sales performance
-	Identify best-selling items within each category
### Limitations
Some rows contained missing data that could not be computed or retrieved from external sources. To maintain accuracy, these rows were removed, resulting in a 4.8% data loss, leaving 11,972 rows available for analysis.
### Conclusion
This project demonstrates an end-to-end data analysis workflow using Microsoft Excel—from data cleaning and transformation to analysis and visualization—resulting in actionable insights that can support effective business decision-making.
